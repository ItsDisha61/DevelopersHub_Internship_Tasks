# Task 5: Personal Loan Acceptance Prediction

## Objective

Predict which bank customers are more likely to accept a personal loan offer using machine learning.

## Approach

* Loaded and explored the Bank Marketing Dataset.
* Performed data exploration on age, job, and marital status.
* Encoded categorical variables and prepared the data.
* Trained a Logistic Regression model.
* Evaluated the model using accuracy and confusion matrix.
* Analyzed customer groups to extract business insights.

## Results and Insights

* The model successfully predicted loan acceptance behavior.
* Job type, age, and marital status influenced customer responses.
* Customer group analysis provided useful insights for targeted marketing strategies.

## Conclusion

This project demonstrates how classification models can help banks identify potential customers and improve marketing decisions through data-driven insights.
