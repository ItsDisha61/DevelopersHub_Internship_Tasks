# Task 3: Customer Churn Prediction

## Objective

Predict whether a bank customer is likely to leave the bank using machine learning.

## Approach

* Loaded and cleaned the Churn Modelling dataset.
* Encoded categorical features such as Geography and Gender.
* Prepared the data and trained a classification model.
* Analyzed feature importance to identify key churn factors.

## Results and Insights

* The model successfully predicted customer churn.
* Customer demographics and account-related features influenced churn behavior.
* Feature importance helped identify factors affecting customer retention.
