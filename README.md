# DevelopersHub Internship Tasks
This repository contains Data Science and Analytics internship tasks completed as part of the DevelopersHub Corporation Internship Program.

## Tasks Included

### Task 1: Exploring and Visualizing a Simple Dataset

* Data loading and inspection using Pandas
* Data visualization using Matplotlib and Seaborn

### Task 2: Credit Risk Prediction

* Data cleaning and exploratory data analysis
* Classification modeling using machine learning

### Task 3: Customer Churn Prediction

* Customer churn classification
* Feature importance analysis

### Task 4: Predicting Insurance Claim Amounts

* Regression modeling using Linear Regression
* Model evaluation using MAE and RMSE

### Task 5: Personal Loan Acceptance Prediction

* Customer behavior analysis
* Classification modeling and business insights

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook
