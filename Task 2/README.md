# Task 2: Credit Risk Prediction

## Objective

The objective of this project is to predict whether a loan applicant is likely to default on a loan using machine learning classification techniques.

## Approach

* Loaded and explored the Loan Prediction Dataset.
* Examined dataset structure using Pandas functions.
* Checked and handled missing values appropriately.
* Performed Exploratory Data Analysis (EDA).
* Visualized important features such as loan amount, education level, and income.
* Encoded categorical variables where required.
* Split the dataset into training and testing sets.
* Trained a Logistic Regression classification model.
* Evaluated the model using Accuracy Score and Confusion Matrix.

## Results and Insights

* The dataset was successfully cleaned and prepared for modeling.
* Visualizations helped identify patterns in income, education, and loan-related features.
* The Logistic Regression model was able to classify loan repayment outcomes.
* Accuracy and Confusion Matrix were used to evaluate model performance.
* The analysis demonstrated how customer financial information can be used to assess credit risk.
