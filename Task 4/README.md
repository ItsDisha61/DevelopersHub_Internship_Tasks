# Task 4: Predicting Insurance Claim Amounts

## Objective

Predict medical insurance charges using a Linear Regression model.

## Approach

* Loaded and explored the Medical Cost Personal Dataset.
* Checked missing values and prepared the data.
* Visualized the impact of age, BMI, and smoking status on insurance charges.
* Encoded categorical features and trained a Linear Regression model.
* Evaluated model performance using MAE and RMSE.

## Results and Insights

* The model successfully predicted insurance charges.
* Smoking status had a significant impact on medical costs.
* Higher age and BMI were generally associated with higher charges.
* MAE and RMSE were used to assess model performance.
