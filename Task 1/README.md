# Task 1: Exploring and Visualizing Iris Dataset

## Objective

The objective of this task is to explore, summarize, and visualize the Iris dataset using Python libraries such as Pandas, Matplotlib, and Seaborn.

## Approach

* Loaded the Iris dataset using Seaborn.
* Examined the dataset structure using:

  * `df.head()`
  * `df.shape`
  * `df.columns`
  * `df.info()`
  * `df.describe()`
* Created visualizations to better understand the dataset:

  * Scatter Plot (Sepal Length vs Petal Length)
  * Histogram (Sepal Length Distribution)
  * Box Plot (Petal Length Distribution)
* Analyzed relationships, distributions, and variation in the data.

## Results and Insights

* The Iris dataset contains **150 records** and **5 columns**.
* No missing values were found in the dataset.
* The scatter plot showed clear relationships between flower measurements and species.
* The histogram illustrated the distribution of sepal length values.
* The box plot helped visualize the spread of petal length and identify potential outliers.
* The dataset was successfully explored and visualized using multiple techniques.
